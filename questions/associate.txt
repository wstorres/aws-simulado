1. Qual serviço da AWS é ideal para armazenamento de objetos altamente escalável e durável?
A) Amazon EBS
B) Amazon S3
C) Amazon RDS
D) Amazon EFS
Resposta: B
Domínio: 2

2. Um arquiteto de soluções precisa executar uma aplicação contêinerizada sem gerenciar servidores. Qual serviço deve ser usado?
A) Amazon EC2
B) AWS Fargate
C) Amazon EMR
D) Amazon S3
Resposta: B
Domínio: 3

3. Qual serviço da AWS fornece entrega de conteúdo global de baixa latência e alta velocidade de transferência de dados (CDN)?
A) Amazon CloudFront
B) AWS Direct Connect
C) Amazon Route 53
D) AWS Elastic Beanstalk
Resposta: A
Domínio: 3

4. Um banco de dados NoSQL totalmente gerenciado na AWS que oferece desempenho de milissegundo de dígito único é o:
A) Amazon RDS
B) Amazon Aurora
C) Amazon DynamoDB
D) Amazon Redshift
Resposta: C
Domínio: 3

5. Qual ferramenta da AWS permite estimar o custo mensal dos recursos que serão implantados?
A) AWS Budgets
B) AWS Pricing Calculator
C) AWS Cost Explorer
D) AWS Trusted Advisor
Resposta: B
Domínio: 4

6. Você é o Arquiteto Chefe da AWS em uma startup. A empresa possui um aplicativo legado que roda na AWS e se conecta ao data center local por meio de uma VPN. Você precisa registrar todo o tráfego da VPN. Qual serviço da AWS pode ser usado nesse caso?
A) Amazon VPC Flow Logs
B) ELB Access Logs
C) AWS CloudTrail Logs
D) Amazon CloudWatch Logs
Resposta: A
Domínio: 1

7. Você deseja lançar um cluster Kubernetes gerenciado de contêineres sem servidor na AWS usando um framework de código aberto. Qual serviço da AWS pode ser usado para atender a esse requisito?
A) SWF
B) EKS
C) Glue
D) ECS
Resposta: B
Domínio: 1

8. Um membro da sua equipe de DevOps consultou você sobre o melhor armazenamento que atenda aos seguintes requisitos: – escalonamento automático; – acesso simultâneo por múltiplas instâncias EC2; – suporte a bloqueio em nível de arquivo. Qual é a sua recomendação?
A) Amazon EFS
B) Amazon S3 Snowball
C) Amazon S3
D) Amazon EBS
Resposta: A
